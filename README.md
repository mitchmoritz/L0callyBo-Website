# l0cally bo website

Website for a creative teen. This is phase 1 of the site: a static creative hub
for socials, portfolio categories, shop planning, and parent-managed booking or
custom order requests.

## What is included

- `index.html` - the website structure and copy
- `styles.css` - the visual design
- `script.js` - the booking request email draft handler
- `assets/creator-collage-hero.png` - generated placeholder hero art
- `assets/art/` - selected artwork imported from Google Drive for the first gallery

## First edits to make

1. Open `script.js`.
2. Replace `parent@example.com` with the parent-accessible email address.
3. Open `index.html`.
4. Replace placeholder wording with Bo's preferred bio, product names, and booking language.
5. Replace `assets/creator-collage-hero.png` with real artwork or a real collage once available.
6. Add more gallery images to `assets/art/` and copy one of the existing gallery cards in `index.html`.

## Phased roadmap

### Phase 1: Launchable hub

- Social links
- Portfolio categories
- Shop preview
- Parent-managed inquiry form
- Safety-first public copy

### Phase 2: Real portfolio

- Add actual drawings, makeup looks, painted jackets, pins, prints, and shirts.
- Add captions, dates, materials, and availability.
- Separate personal/social content from sellable/booking content.

### Phase 3: Shop setup

- Choose Etsy, Shopify, Square, PayPal, or another checkout tool.
- Add product pages or external shop links.
- Define shipping, pickup, payment, returns, and custom order rules.

### Phase 4: Booking workflow

- Replace mailto with a real form service.
- Route requests to a parent-accessible inbox.
- Add service types, time estimates, price ranges, and event date fields.

### Phase 5: AI agent support

- Use agents to sort requests by type.
- Draft parent review replies.
- Track follow-up status.
- Connect to scheduling only after an adult reviews each request.

## Teen creator safety notes

- Do not publish home address, school, personal phone number, or exact location.
- Keep email and payment accounts parent-managed.
- Confirm every booking, commission, and sale through an adult before accepting money.
- Keep the public site focused on creative work, not private life details.
